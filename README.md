# KeyLogger-Python-With-Discord-Webhook

ONLY FOR EDUCATION PURPOSES.

This is a simple keylogger that uploads to discord via webhook.

To make a webhook, you need a discord server.
Create the server and then create a channel. Go to that channel setting and find integrations. Then press webhook and create one. Finally name the webhook and copy the url. DONT SHARE THE URL OR ANY ONE WHO HAS ACCESS TO IT CAN SPAM YOUR CHANNEL
